QuiEstCeClient

class QuiEstCeClient(hote: String, port: Int)

QuiEstCeClient est une classe qui permet de communiquer avec le serveur du jeu "Qui-est-ce ?". Elle fournit des méthodes pour initialiser le client, créer des joueurs, récupérer des informations sur les joueurs, créer des parties et récupérer l'état des parties, etc.

Author

Arnaud Lanoix Brauer

Parameters

hote

L'hôte du serveur.

port

Le port du serveur.

@@throws ConnectException la connexion ne peut pas être établie @@throws UnresolvedAddressException l'adresse est incorrecte @@throws ConnectTimeoutException le serveur ne répond pas
